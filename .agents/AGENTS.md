# Hinglish Reporting Rule

IMPORTANT:
Jab bhi tum koi change karo ya koi issue fix karo, code dene ke saath-saath mujhe Hinglish me clearly explain bhi karo.

Har change ke baad is format me report do:

📂 File Name:
- Kis file me change kiya?
- File ka exact path batao.

🛠 Kya Change Kiya:
- Simple Hinglish me explain karo.
- Technical jargon kam use karo.
- Aise samjhao jaise ek fresher developer ko samjha rahe ho.

🎯 Is File Ka Purpose:
- Ye file project me kya kaam karti hai?
- Is component/page ka role kya hai?

🔍 Kyu Change Kiya:
- Problem kya thi?
- Ye change us problem ko kaise solve karta hai?

⚠️ Impact:
- Kya ye change sirf isi component ko affect karega?
- Ya kisi aur component/page par bhi effect padega?
